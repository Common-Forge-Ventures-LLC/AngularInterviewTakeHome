Instructions

Your goal is to create an application where we can see a list of employees and edit their data. This list should allow for the selection of an employee, which would then show more details for him/her. With an employee selected, we should be able to edit his/her data. By clicking save we should see the changes reflected in the employee list. We should also have the ability to delete an employee and add an employee.


Objectives

  1. Display selectable list of all employees. This list will show the employee names and positions.
  2. A selected employee should then show additional details in the card to the right.
  3. All employee Data should be able editable.
  4. We should be able to create a new Employee, that also shows up in the left hand list once created.
  5. We should be able to remove an employee, that is also removed from the left hand list after deletion.

Additional info and Rules

  • A employee route and blank component has already been created for you.
  • The '/employee' route is where the user would navigate to see both the employee list and employee details.
    No other route creation is needed.
  • The employee data can be found in the employee.service.ts file.
    Please keep the data in this file.
  • Must use/create at least 1 reusable component.
  • The list of employees and the employee details must be in separate components.
